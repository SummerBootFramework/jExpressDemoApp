Welcome to jExpress Demo Application

This demo will demonstrate the following functional and non-functional aspects provided.

Functional Features


  • RESTFul service - @Controller with lifecycle hooks (AppLifecycleHandler), include Idle event callback
  • gRPC service - @GrpcController
  • WebSocket - OTT (One Time Ticket) based authentication with chat room aand large file upload handlers
  • Create PDF
  • PDF to images
  • File Transfer - Upload
  • ile Transfer - Download
  • LDAP integration via config file or customized auth via @Service(binding = Authenticator.class) or for gRPC: @Service(binding = {Authenticator.class, ServerInterceptor.class})
  • IOC - @Service and AlternativeName
  • Scheduled Tasks — @Scheduled

Non-Functional Features


  • Performance: RESTful Web Services (JAX-RS) with Non-blocking I/O (powered by Netty Reactor)
  • Bearer authentication (also called token authentication) - login/logout
  • Role Based Authorization - @RolesAllowed, @PermitAll and @DenyAll
  • Load Balancer health check - @Ping
  • Graceful Shutdown and @Daemon
  • Health Check - @HealthCheck and @RequiresHealthCheck
  • Protected logging - @@Log
  • Logging After Response Sent
  • Log tracking - X-Reference in response header
  • Performance tracking - POI (Point of Interest)
  • CLI — A/B/Mock Mode
  • CLI — List and Check Duplicate Error Codes via @Unique
  • Auto-Generated configuration Files - @Config
  • Auto-Protected configuration sensitive data
  • Configuration Hot-Reload
  • Auto Email Alert when application start, stop, encounter errors, config file changed, health check and status changed
